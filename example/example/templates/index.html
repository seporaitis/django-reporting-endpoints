Welcome to Example app

This app is designed to show how to integrate django-reporting-endpoints into your project.

You will not believe how easy it is!

This page is already set-up to trigger and report a CSP violation. However, some browsers only send reports when the page is served via HTTPS. You can use Ngrok, or similar, to create a https proxy and open the page via https. (e.g. `ngrok http 8000`).

Read more in the documentation.
